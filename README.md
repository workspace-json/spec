workspace-json specification (v0.1)

This repository contains the canonical `workspace.json` specification and schema.
It is the authoritative source for the spec and intended for review by standards
audiences (AAIF, implementers, and integrators).

Quick links
- Spec document: [spec.mdx](spec.mdx)
- JSON Schema: [schema/v1.json](schema/v1.json)
- Examples: [examples/](examples/)
- Governance: [GOVERNANCE.mdx](GOVERNANCE.mdx)

Overview
This repo aims to be small, reviewable, and machine-friendly. The rendered
documentation is hosted at https://workspacejson.dev (the website repository).

If you are evaluating the spec for adoption or contribution, see `CONTRIBUTING.md`
for the RFC process and `CODE_OF_CONDUCT.md` for community guidelines.

Contact
Governance and maintainership details live in `GOVERNANCE.mdx`. For mailing
lists and governance contact, see the governance file.
